This project is an external ESP/aimbot for Knives Out. It includes boxes, skeletons, snaplines, and an aimbot, along with a pretty clean-looking ImGui menu.
I can’t guarantee it still works today since it hasn’t been updated in almost two years—there’s a good chance the signatures are outdated. 
But if they haven’t changed, it should still run just fine.
